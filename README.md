
# Harmonia - Psychoterapia Online

Strona typu landing page dla psychoterapeuty (mgr Katarzyna Organ).

## Technologie
- **React 19**
- **Vite** (szybki build)
- **Tailwind CSS** (estetyczny wygląd)
- **Lucide React** (ikony)

## Instalacja lokalna
1. Pobierz pliki.
2. Zainstaluj zależności: `npm install`
3. Uruchom projekt: `npm run dev`

## Publikacja
Najlepiej opublikować za pomocą Vercel lub Netlify, łącząc bezpośrednio repozytorium z GitHub.
